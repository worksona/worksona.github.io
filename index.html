Quick Start Guide

Get up and running with Worksona.js in minutes

1. Include the Library

<script src="worksona.min.js"></script>

2. Initialize with API Keys

// Create a new Worksona instance const worksona = new Worksona({ apiKeys: { openai: 'your-openai-api-key', anthropic: 'your-anthropic-api-key', google: 'your-google-api-key' }, debug: true, controlPanel: true });

3. Define an Agent

const agent = { id: 'customer-support', name: 'Sarah', description: 'Friendly customer support specialist', traits: { personality: ['empathetic', 'patient', 'solution-oriented'], knowledge: ['product catalog', 'return policies'], tone: 'warm and professional' }, config: { provider: 'openai', model: 'gpt-3.5-turbo', temperature: 0.7, systemPrompt: 'You are Sarah, a friendly customer service rep...' } }; // Load the agent await worksona.loadAgent(agent);

4. Chat with Your Agent

// Send a message to the agent and get a response const response = await worksona.chat('customer-support', 'I need help with my recent order'); console.log(response);
